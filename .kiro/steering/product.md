# Product Overview

This is a browser-based game workshop project called "Legend of Kiro" - a Zelda-inspired dungeon crawler where players navigate a grid-based maze to collect AWS items while avoiding ghosts.

## Core Gameplay
- Grid-based movement using arrow keys
- Collect all items to win the level
- Avoid ghost enemies that patrol the dungeon
- Player has 3 health points and respawns on hit
- Retro pixel art aesthetic with Kiro branding

## Target Audience
Workshop participants at AWS Re:Invent learning to build games with AI assistance. The game serves as a hands-on example for developers of all skill levels to learn game development fundamentals in a collaborative 2-hour session.
